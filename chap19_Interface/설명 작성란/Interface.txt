인터페이스(Interface)란?
	클래스와 클래스간의 계약을 정의하는 추상 데이터 유형
	클래스의 행동을 정의하고, 클래스가 특정 메서드를 반드시 구현해야 함을 강제한다.
	메서드 선언만 포함한다. -> 메서드의 이름, 매개변수 및 반환 유형만 선언하고 메서드의 구현 내용은 제공하지 않음
	추상 메서드만 포함한다. -> 모든 메서드는 추상 메서드로 선언되어있기 때문에 메서드 본문이 없고 메서드를 호출하면 해당 인터페이스를 구현하는 클래스에서 정의된 메서드 실행
	다중 상속을 지원한다. -> 클래스는 하나의 클래스만 상속할 수 있지만, 인터페이스는 다수의 인터페이스를 구현하고 여러 인터페이스에서 정의된 메서드를 동시에 사용할 수 있다.

예제)
	public interface MyInterface{
		void myInterfaceMethod();
	}
