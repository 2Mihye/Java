(if / else if / else) 조건문이란?
	위 문장을 사용하여 조건에 따라 다른 코드블록 실행
	주어진 조건을 평가하고 참일 경우 특정 코드를 실행하고 거짓일 경우 다른 코드를 실행함.

if (조건문) {
	참(true)이면 실행되는 코드
} 거짓(false)라면 종료



if-else : 프로그램 조건에 따라 두 가지 서로 다른 코드불록 중 하나를 실행하는 제어구조
		 만약 if문의 조건이 참이면 if 내에 있는 블록이 실행되고 조건이 거짓이면 else 블록이 실행됨.
		 이를 통해 두 가지 상황에 따라 다른 동작을 수행할 수 있음.

if (조건문){
	참(true)이면 실행되는 코드
} else {
	거짓(false)일 때 실행되는 코드
}


if-else if-else : 여러 개의 조건을 순차적으로 평가하고, 그 중 하나의 조건에 해당하는 코드블록을 실행하는 구조
			모든 조건을 차례대로 검사하며, 첫 번째로 참이 되는 조건에 해당하는 코드블록만 실행되고 나머지 조건은 무시된다.

			여러개의 if 조건을 연속적으로 검사하면서, 첫 번째로 참이 되는 조건의 코드블록을 실행하고 else if 는 이전 조건이 거짓일
			때만 실행되며, 여러개의 조건을 추가할 수 있다.
			마지막 else 블록은 이전 모든 조건이 거짓일 때 실행
			
if(조건1) {
	조건1이 참일 때 실행되는 코드
} else if (조건2) {
	조건2가 참일 때 실행되는 코드
} else {
	위의 모든 조건이 거짓일 때 실행되는 코드
}
