메모리란?
	컴퓨터가 정보를 저장하고 처리하는 공간
	메모리는 세 가지 종류로 나뉨
	1. Static Memory(스태틱 메모리) : 컴퓨터가 프로그램의 기본 정보와 데이터를 저장하는 곳으로 그 정보는 프로그램이 실행되는 동안 계속 유지
								  프로그램 안에 공유하는 중요한 정보를 여기에 저장함
	2. Heap Memory(힙 메모리) : 컴퓨터가 프로그램이 필요할 때마다 새로운 데이터를 저장하는 곳
							 이 데이터는 객체라고 불리는데 프로그램 실행 중 계속해서 생기고 사라짐.
							 예를 들어 사용자의 정보나 게임 캐릭터 정보를 저장하는데 사용
	3. Stack Memory(스택 메모리) : 프로그램이 함수나 매서드를 실행할 때 사용하는 곳
								각 함수에는 자체적인 공간이 있어서 함수가 실행되는 동안 필요한 정보를 저장하고, 함수가 끝나면 해당 정보는 지워짐
								

구분되어 있는 이유?
	프로그램이 정보를 보관하고 처리하는 방법을 이해하는데 도움을 줌
	메모리를 효율적으로 사용하면 프로그램이 빠르고 안정적으로 작동 됨
	메모리를 잘 관리하면 프로그램이 크래시되는 걸 방지할 수 있음.
		크래시? : 컴퓨터 프로그램 시스템이 예상하지 못한 오류 또는 결함으로 인해 비정상적으로 종료되거나 작동하지 않은 상태
