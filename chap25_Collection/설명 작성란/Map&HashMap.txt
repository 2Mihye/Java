Map과 HashMap
Map :
	인터페이스
	키(key) - 값(value) 의 쌍으로 데이터를 저장하는 자료 구조이고 각 키는 유일하며 해당 키와 연결된 값을 검색하거나 변경할 수 있다.
	
HashMap :
	Map 인터페이스를 구현한 Java Class
	해시 테이블을 기반으로 키(key) - 값(value) 쌍을 저장하고 관리한다.
	키(key)와 값(value)을 저장하는데 사용한다.
	키는 중복될 수 없으며 유일해야 한다.
	순서를 보장하지 않고, 검색, 추가, 제거 작업이 빠르게 수행된다.
	
	*해시 테이블이란? 데이터를 효율적으로 저장하고 검색하기 위한 테이블 자료 구조
				 해시 함수를 사용하여 데이터를 고유한 해시코드로 매핑(감쌈)
				 이 해시코드를 사용해서 데이터를 저장하고 검색한다.
