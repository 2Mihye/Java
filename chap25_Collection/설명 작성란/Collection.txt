Collection이란?
	데이터를 저장, 관리 및 조작하기 위한 인터페이스와 클래스의 모음을 나타냄
	컬렉션은 클래스들이라 데이터를 구조화하고 다양한 방법으로 접근 조작할 수 있는 다양한 자료 구조를 제공
	자주 사용하는 컬렉션 인터페이스 및 클래스가 있다.
		- List
		- Set
		- Map
	
	메서드
	   sort() : 오름차순
	reverse() : 내림차순
 	 remove() : 지정된 요소를 컬렉션에서 제거한다.
		add() : 요소를 컬렉션에 추가한다.
	isEmpty() : 컬렉션이 비어 있는지 여부 확인
	 addAll() : 다른 컬렉션의 모든 요소들을 현재 컬렉션에 추가
   contains() : 컬렉션이 지정된 요소를 포함하고 있는지 여부를 확인함
      clear() : 모든 요소 제거
    toArray() : 컬렉션의 요소를 배열로 반환한다.
   iterator() : 컬렉션 요소에 반복하는데 사용되는 반복자를 반환
